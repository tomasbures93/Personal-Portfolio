# Portfolio

Portfolio is a full-stack project for building my personal portfolio website.  
The goal of this project is to create a place where I can present my current and future projects, showcase the technologies I work with, and publish blog posts about interesting IT topics and coding-related news.

This project is mainly a learning project. A major focus is building it with clean and maintainable backend architecture by following Clean Architecture principles. It also includes structured error handling through global exception handling and the Result Pattern to provide clear and consistent API error messages.

## Current Status

The project is currently in Phase 1.
